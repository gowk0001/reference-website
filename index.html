HTML Regions

<header>

This region is at the top of a webpage and contains site-wide information. This typically includes a logo, search function, and navigation options.

<nav>

The nav or navigation region contains a menu used to navigate through a website's content.

<body>

The body region contains all of the content on a webpage.

<main>

The main region is a contextual section that contains all of a webpage's main content.

<footer>

Similar to the header, the footer region also cotains site-wide information. This can include copyright information, privacy statements, disclaimers, and occasionally secondary navigation.

An unordered list, where the order of items within does not matter. Contains li (list items) as its children.

A description list, which groups title and definitions. Contains dt (description title) and dd (description definitions) as its children.

CSS Properties

margin

Sets the margin, or the outside spacing, of an element. Can be specified to specific areas around the element using margin-top, margin-bottom, margin-left, and margin-right.

Example

margin: 5px;

margin: 5px 20px;

margin-left: 10px;

padding

Sets the padding, or the spacing between an element's content and its border. Can be specified to specific areas around the element using padding-top, padding-bottom, padding-left, and padding-right.

Example

margin: 5px;

padding: 5px;

margin: 5px;

padding: 5px 20px;

margin: 5px;

padding-right: 20px;

width

Sets the precice width of an element.

height

Sets the precise height of an element.

max-width

Sets the maximum width of an element.

background-color

Sets the color of an element's background.

background

Sets an element's background. This can be a color as well as an image.

border

Sets a border around an element. Can be specified with weight as well as type; i.e. solid, double, dashed. Can also be set to specific areas of an element by using border-bottom, border-left, border-right, and border-top.

Example

border: solid 5px black;

border-left: dashed 5px blue;

border-bottom: double 2px red;

color

Sets the color of the text within an element.

font-family

Sets the font family for the text within an element.

font-size

Sets the font size for the text within an element.

font-weight

Sets the font weight for the text within an element.

line-height

Sets the vertical spacing between the lines of text within an element.

text-align

Sets the alignment for the text within an element; i.e. left, center, right, and justify.

text-decoration

Sets the style and color of underlined text within an element; i.e. none, overline, underline, line-through, etc.

text-transform

Sets the capitalization and casing for text within an element; i.e. none, uppercase, lowercase and capitalized.

Flexbox

Flexbox allows you to arrange elements in a flexible and dynamic way by aligning and distributing items within a container, regardless of their size, and adjust their spacing, direction, and order.

To create a Flexbox layout, you need to define a container element and specify its display property as flex or inline-flex. Once the container becomes a flex container, its child elements become flex items, which can be manipulated using a set of properties.

display

This property defines the display type of the flex container. It can be set to either flex or inline-flex.

flex-direction

This property sets the direction of the flex items within the container. It can be set to row, row-reverse, column, or column-reverse.

flex-wrap

This property controls whether the flex items should wrap to multiple lines or not when they exceed the width or height of the flex container. It can be set to nowrap (default), wrap, or wrap-reverse.

justify-content

This property aligns the flex items along the main axis. It can be set to flex-start, flex-end, center, space-between, space-around, or space-evenly.

align-items

This property aligns the flex items along the cross axis. It can be set to stretch, flex-start, flex-end, center, or baseline.

align-content

This property aligns the lines of flex items within the flex container when there is extra space on the cross axis. It only applies when the flex items are on multiple lines. It can be set to flex-start, flex-end, center, space-between, space-around, or stretch.

align-self

This property aligns a single flex item along the cross axis, overriding the align-items value for that item. It can be set to auto, flex-start, flex-end, center, baseline, or stretch.

CSS Grid

CSS Grid allows you to arrange elements within the rows and columns of a grid, by specifying the size and position of each element.

To create a grid, you need to define a container element as a grid, you can then create rows and columns, and specify the size and position of grid cells.

display: grid

Defines the container element as a grid.

grid-template-rows

Defines the size of the rows in the grid.

grid-template-columns

Defines the size of the columns in the grid.

grid-template-areas

Defines a grid template using named grid areas.

grid-gap

Defines the size of the gap between rows and columns in the grid.

justify-items

Defines how the grid items align along the horizontal axis.

align-items

Defines how the grid items align along the vertical axis.

justify-content

Defines how the grid items align within the grid.

Media Queries

Media queries allow you to apply different styles to a webpage based on the screen size being used. This means that the same webpage can be optimized for a variety of devices, such as desktops, tablets, and smartphones, ensuring that the user experience is optimized for each device.
